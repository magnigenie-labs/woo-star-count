This plugin allows your visitors to see a visual overview of total stars/review given to a specific product and also allows visitors to mark a review as thumbs up or thumbs down.

== Description ==

Review Stars Count For WooCommerce enables your customers to see a visual overview of all the rating stars give to specific products and the plugin also allows them to mark any reviews on product page to helpful or not helpful by choosing thumbs up or thumbs down.

== Installation ==
1. Upload "woo-star-count.zip" file to your "/wp-content/plugins/" directory.
2. Once uploaded go to your WordPress admin and navigate to plugins.
3. Once you are on plugins screen then you can see Review Stars Count For WooCommerce listed there.
4. Just activate and start playing with the plugin configurations.

== Frequently Asked Questions ==
= Why should I use this plugin? =
We all like to know more insights about the products before we actually purchase and thus this plugin will help your customers to know the exact number of rating/reviews given to a specific product and accordingly judge their purchase.
